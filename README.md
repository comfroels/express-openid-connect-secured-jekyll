# Express OpenID Connect Secured Jekyll

An example of securing a Jekyll powered website with Express using OpenID Connect.

## Usage

- `cp .env.example .env`
- Add configuration in **.env**
- `npm install`
- `npm start`

## Requirements

- Node.js
- Ruby v2
- `gem install bundler`
